Element.prototype.attributes
    - 프로퍼티로 모든 어트리뷰트 노드를 취득할 수 있다.
    - getter만 존재하는 읽기 전용 접근자로 NamedNodeMap을 반환한다.
  
    Element.prototype.getAttribute/setAttribute
    - attributes 프로퍼티를 통하지 않고 요소 노드에서
      메서드를 통해 직접 HTML 어트리뷰트 값을 취득하거나 변경할 수 있다.
  
    Element.prototype.hasAttribute : 특정 HTML 속성의 존재 여부 확인
    Element.prototype.removeAttribute : 특정 HTML 속성의 존재 삭제
